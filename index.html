CPU Efficiency by MHz

CPU Single score per MHz (higher = more efficient)

CPU Single-core score divided by maximum CPU frequency. Higher values mean the CPU delivers more performance per clock cycle. Formula: CPU Single ÷ CPU Max MHz.
—

GPU Efficiency by MHz

GPU score per MHz (higher = more efficient)

GPU score divided by maximum GPU frequency. Higher values mean the GPU delivers more performance per clock cycle. Formula: GPU Score ÷ GPU Max MHz.

Bottleneck Ratio

Ratio = CPU Multi ÷ GPU Score. <1 CPU is the limiter. >1 GPU is the limiter.

GPU vs CPU Bottleneck

<1 CPU bottleneck ~1 Balanced >1 GPU bottleneck
CPU Multi ÷ GPU Score. A balanced system has a ratio near 1.0. Below 1.0 means the GPU outruns the CPU (CPU bottleneck). Above 1.0 means the CPU outruns the GPU (GPU bottleneck).
